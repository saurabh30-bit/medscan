# MedScan: Medicine Authenticity Checker

**MedScan** verifies medicine authenticity by cross-referencing batch numbers, manufacturer names, and date integrity against known pharmaceutical standards. Built to combat the global counterfeit drug epidemic that affects 10% of medicines in developing countries.

## ✨ Features
- **Manufacturer Verification**: Checks against 25+ recognized pharmaceutical companies.
- **Batch Number Validation**: Validates format against standard pharmaceutical coding patterns.
- **Date Integrity**: Checks shelf life, expiry status, and flags anomalies.
- **100% Client-Side**: All checks run locally — no data transmitted.

## 🚀 Tech Stack
HTML5, Vanilla CSS3, Vanilla JavaScript (ES6+)

## 💻 Getting Started
```bash
git clone https://github.com/saurabh30-bit/medscan.git
cd medscan && python -m http.server 8000
```
